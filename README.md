## 프로젝트 빌드 및 실행방법

1. 레파지토리 클론 다운받기

```
git clone https://github.com/suyeon-hong/calulator.git
```

2. 패키지 설치하기

```
yarn install
```

3. .env 파일 생성하기 - [currencylayer](https://currencylayer.com/) api 이용

```
REACT_APP_API_KEY = ''
REACT_APP_BASE_URL = ''
```

4. 서버 실행하기

```
yarn start
```
